medialog.tablebehavior Installation
--------------------------------

To install medialog.tablebehavior using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``medialog.tablebehavior`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        medialog.tablebehavior
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
